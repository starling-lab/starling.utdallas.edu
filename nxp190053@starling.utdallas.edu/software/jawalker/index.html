    JA-Walk-ER

    Just-another Walk-ER with a GUI.


    Docs View Source

    Copyright © 2017-2019 StARLinG Lab. This program comes with absolutely no warranty. This is free software, available under the terms of the GPL-3.0.


    This is Just-another implementation of Walk-ER. Because of dependencies which need license, the original code no longer supports a GUI. This code is built on top of the original code to provide a GUI.

    The original source code and TeX for “User Friendly Automatic Construction of Background Knowledge: Mode Construction from ER Diagrams.” can be found here.
